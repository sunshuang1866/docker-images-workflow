# 修复摘要

## 修复的问题
无需代码修改。CI 失败为基础设施误报（infra-error）。

## 修改的文件
无

## 修复逻辑
CI 分析报告明确指出该失败为 `infra-error`，置信度中。失败原因是 eulerpublisher 的 appstore 发布规范预检工具在扫描变更文件时，将仓库根目录的 `README.md` 误纳入应用镜像发布规范校验流程，对其路径进行格式校验时判定路径不符合预期（期望 `/README.md`）。该检查工具设计用于验证应用镜像目录结构，对仓库根级文档文件的路径解析存在兼容性问题。

PR #3153 仅修改了 `README.md` 中基础镜像 Tags 列表的文档内容，未涉及任何应用镜像构建文件或目录结构变更，与 CI 检查逻辑无关。此问题应由 CI 基础设施团队在 eulerpublisher 的预检逻辑中修复，使其跳过仓库根目录的非应用镜像文件。

## 潜在风险
无。未对任何源代码进行修改。
