## 修复逻辑
CI 失败分析报告明确判定为 `infra-error`，根因是 `repo.openeuler.org` 在 CI 构建时段返回 HTTP/2 流错误（Curl error 92: `INTERNAL_ERROR`），导致 `dnf install` 下载依赖包失败。该错误与 PR #2991 的代码变更无关，属于上游镜像源临时性服务端/网络协议层问题。

**建议操作**：触发 CI 重试（re-run），该临时性基础设施问题大概率在重新构建时不再出现。
